Infer.NET 2.6 Release

Welcome to Infer.NET!

Infer.NET is a library for probabilistic inference in graphical models providing state-of-the-art message-passing algorithms, graph algorithms, and linear algebra routines needed by modern machine learning applications. It is available under a non-commercial license.

Documentation

Documentation is available in online form at http://research.microsoft.com/infernet. It includes a user guide, tutorials and examples, and code documentation.
